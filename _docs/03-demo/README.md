# Demo Documentation

This folder contains all materials for demonstrating the application.

## Files

- **DEMO_BY_FEATURE.md** - **Feature-by-feature demo guide with persona benefits** (Recommended for mixed audiences)
- **DEMO_BY_PERSONA.md** - Tailored demo guides for specific stakeholders (Site Planners, Project Managers, Civil Engineers, Executives)
- **DEMO_SCRIPT.md** - Generic presentation-ready demo script with talking points
- **DEMO_GUIDE.md** - Technical walkthrough for demos
- **DEMO_PROPERTY_TYPES.md** - Explanation of property types (flat, hilly, constrained)
- **TESTING_CHECKLIST.md** - Comprehensive testing procedures
- **PROJECT_COMPLETE.md** - Project completion summary

## Quick Links

- **Preparing a demo?** → Start with `DEMO_BY_FEATURE.md` (feature-by-feature with persona benefits) or `DEMO_BY_PERSONA.md` (persona-specific)
- **Mixed audience?** → Use `DEMO_BY_FEATURE.md` - shows each feature and how all personas benefit
- **Single persona audience?** → Use `DEMO_BY_PERSONA.md` - tailored for specific roles
- **Need test procedures?** → See `TESTING_CHECKLIST.md`
- **Understanding property types?** → See `DEMO_PROPERTY_TYPES.md`
